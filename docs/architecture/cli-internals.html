CLI Internals

Design Philosophy

The Rendro CLI is a zero-dependency Node.js script. It uses only Node.js built-in modules — fs, path, crypto, and http — to avoid dependency bloat and ensure predictable behavior across platforms.

The entire CLI fits in a single 7KB file. It's distributed both as an npm package (with zero runtime dependencies) and as a standalone script downloadable via curl.

Hash-Based Diffing

Before uploading, the CLI computes MD5 hashes of local files and compares them against server-side ETags. Files with matching hashes are skipped, dramatically reducing upload time for large documentation sets.

The diffing algorithm works in three phases: (1) fetch existing file list from server, (2) walk local directory and compare hashes, (3) upload changed/new files and soft-delete removed files.
