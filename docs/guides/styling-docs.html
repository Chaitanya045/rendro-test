Styling Your Docs

CSS Approach

Rendro serves raw HTML, so all styling must be inline or included via <style> tags within each HTML file. External stylesheets via CDN links are supported and recommended for consistency across pages.

Popular choices include: Tailwind CSS (via CDN), Pico CSS, Water.css, or a custom design system. The Rendro web interface uses a separate CSS scope and won't interfere with your doc styling.

Dark Mode

If your docs include dark mode support, use prefers-color-scheme media queries or a class-based toggle. The Rendro sidebar and topbar have their own dark mode toggle that won't conflict with your doc styling.
