Migration Guide

From Other Platforms

Migrating to Rendro from other documentation platforms is straightforward. Rendro accepts raw HTML, so the primary task is exporting your existing content as HTML files.

For most platforms, you can use built-in export tools or write a simple script to convert your content to HTML. The Rendro CLI handles the upload.

From GitBook

GitBook exports can be converted to HTML using their API or by downloading the static site export. Organize the HTML files by section, matching your desired sidebar tree structure.

From Confluence

Confluence spaces can be exported as HTML via the Space Tools → Export menu. The export includes attachments and preserves the page hierarchy. Extract the HTML files and organize them into folders matching your space structure.

From Notion

Notion pages can be exported as HTML + CSV via the Export menu. The export includes subpages as separate files. Use the folder structure to mirror your Notion sidebar.
