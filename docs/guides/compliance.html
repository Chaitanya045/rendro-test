Compliance

Overview

This page covers compliance in the Rendro documentation platform. For more details, see the related sections in the sidebar.

The guides section provides comprehensive information about configuring, using, and extending Rendro for your team's documentation needs.

Configuration

Configure compliance through environment variables or the Rendro configuration file. All settings are optional with sensible defaults.

# Example configuration for compliance
export RENDRO_COMPLIANCE_ENABLED=true

Best Practices

When working with compliance, follow these guidelines to ensure optimal performance and maintainability.

  • Always test changes in a staging environment first
  • Document your configuration choices for team members
  • Monitor for changes in behavior after updates
