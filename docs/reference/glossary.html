Glossary

Terms

Organization (Org) — A namespace for documentation, typically derived from email domain. Each org has its own API keys and blob storage prefix.

API Key — A per-organization secret used to authenticate CLI operations. Format: rendro_ followed by 32 random characters.

Soft Delete — A deletion strategy that marks files as hidden rather than removing them from storage. Files remain accessible via direct URL.

Tree API — The REST endpoint that provides folder/file listings for the lazy-loading sidebar tree.

Hash-Based Diffing — The CLI's approach to detecting changed files by comparing MD5 hashes of local content against server ETags.

Commentor — The inline commenting widget injected into doc pages. Supports text selection, threaded replies, and real-time updates via Convex.
