Build Your First Doc Site

Note: This tutorial assumes you have a Google account and Node.js 18+ installed.

Step 1: Install the CLI

Start by installing the Rendro CLI. You can use npm or download the standalone script.

npm install -g rendro
# or
curl -sL https://raw.githubusercontent.com/Chaitanya045/rendro/main/bin/rendro.mjs -o rendro
chmod +x rendro

Step 2: Create Your First Doc

Create a docs directory with an index file. Use the sample template below to get started with a styled page.

mkdir -p docs
echo '

My Docs




Welcome to My Docs


This is my first documentation page.


' > docs/index.html

Step 3: Push Your Docs

Sign in to the Rendro web interface, create your organization, and copy the API key. Then push your docs.

export RENDRO_API_KEY=rendro_xxx
rendro push --source ./docs --org my-org

Step 4: Set Up CI/CD

Add a GitHub Actions workflow to automatically push docs on every commit. This ensures your documentation stays in sync with your code.

Create .github/workflows/sync-docs.yml with the sync script. Store your API key as a repository secret.
